# Straddie Capsule Surge Lab

A multi-page static concept site for a theoretical inland capsule hotel, civic simulation lab and island compute layer on Minjerribah / North Stradbroke Island.

The public frame is brave but site-neutral. It argues that Straddie needs a small spare room for the future: beds, civic AI, idle capsule GPU-hours, health-surge logistics, local training and co-op value. It does not nominate an address. It quantifies the kind of value a suitable space could create and why a space should be found.

## Pages

- `index.html` - gateway and public thesis
- `metrics.html` - value metrics and simple scenario model
- `funding.html` - not-for-profit capsule-hotel funding model, access blocks and simulation proof layer
- `compute.html` - idle GPU pool, disaster kiosk, noticeboard, digital twin and supercompute rack pathway
- `simulations.html` - civic simulation, AI and clean-city uses
- `health-surge.html` - health-surge support, Aura of Intelligence and privacy boundaries
- `boundaries.html` - public/private and site-selection limits
- `sources.html` - source trail used for the draft
- `builders/` - browser-local Markdown builders

Shared JavaScript renders the header, footer, floating back-to-top button and bottom previous/next page navigation.

## Local Use

Open `index.html` in a browser, or serve the folder with a small local server for testing.

No backend is required. Builder forms run in the browser and generate Markdown locally.
